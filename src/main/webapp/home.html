Grading Management System


The system presented is used to arrange,sort,change and view the various grades of students using the REQUIRED PARTICULARS from the users.It's key abilities are listed


  • Listing the grades of students in a specific order(top-bottom).

  • Allow User to enter a specific grade to particular student.

  • Ability to display 'subject-wise rank holder' on a Particular Subject.

  • Admin can define the score range for Grading calculation.

  • User can view a particular student's Result.
